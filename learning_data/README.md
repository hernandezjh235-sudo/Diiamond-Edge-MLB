# learning_data

Put saved MLB logs here so the app can run faster and keep learning.

Recommended files:
- `processed/pitcher_game_logs.csv`
- `processed/batter_game_logs.csv`
- `processed/underdog_lines_history.csv`
- `processed/results_history.csv`
- `processed/player_id_map.csv`

The app can still pull live MLB logs if these are empty.
